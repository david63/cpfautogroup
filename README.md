## Minimum Requirements
* phpBB 3.2.0
* PHP 5.4

## Note
This extenstion requires the Auto Group extension to be installed and enabled.

## Install
1. [Download the latest release](https://github.com/david63/cpfautogroup/archive/3.2.zip) and unzip it.
2. Unzip the downloaded release and copy it to the `ext` directory of your phpBB board.
3. Navigate in the ACP to `Customise -> Manage extensions`.
4. Look for `CPFs for Auto Groups` under the Disabled Extensions list and click its `Enable` link.

## Usage
1. Navigate in the ACP to `Users and Groups -> Groups -> CPF autogroup settings`.
2. Apply the settings that you require.

## Uninstall
1. Navigate in the ACP to `Customise -> Manage extensions`.
2. Click the `Disable` link for `CPFs for Auto Groups`.
3. To permanently uninstall, click `Delete Data`, then delete the cpfautogroup folder from `phpBB/ext/david63/`.
